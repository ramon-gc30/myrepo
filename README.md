# myrepo
Repositorio de ejemplo

Primero se descarga la versión más reciente del repositorio remoto (ubicado en GitHub), a través de `pull`

Y después se envían los cambios efectuados en el repositorio local (situado en RStudio) mediante `push`

Recuerda:

1. Guarda el archivo de forma local `ctrl + s`
2. Inserta confirmación (commit)
3. Envía los cambios con `push`
